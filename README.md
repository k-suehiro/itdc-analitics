# ITDC Analytics

BigQuery を使った業務集計レポートツールです。Google Apps Script (GAS) の Web アプリとして動作し、期・集計軸・集計値を選択して実行すると、BigQuery から集計結果をテーブル表示します。

## 機能

- **対象期の選択** — 39〜42期（2022年7月〜2026年4月）
- **上期 / 全期 / 下期の切り替え** — 集計後にクライアント側で期間を絞り込み（再集計不要）
- **集計値の切り替え** — 時間（h） / 金額（千円）
- **集計軸**
  - 作業項目
  - 社員
  - 品名
  - 顧客
  - 請求先部門
  - 研究開発
  - 研究開発（品名）
  - 役務稼働率（社員）
  - 稼働率（社員）
- **TD製番トグル** — TD製番を含む / 除外をクライアント側で切り替え（再集計不要）
- **インクリメンタルサーチ** — 行名でリアルタイム絞り込み
- **列ヘッダーソート** — 任意の列で昇順 / 降順ソート
- **グラフ表示** — Chart.js による棒グラフ / 折れ線グラフ（上位10件 / 全件切り替え可）
- **CSV ダウンロード** — 表示中のデータを BOM 付き UTF-8 CSV で保存
- **SQL 表示** — 実行した BigQuery SQL を確認可能

## ファイル構成

| ファイル | 説明 |
|---|---|
| `コード.gs` | GAS サーバーサイド。BigQuery SQL 生成・実行 |
| `index.html` | フロントエンド UI（HTML / CSS / JS） |
| `appsscript.json` | GAS プロジェクト設定（BigQuery API 有効化など） |

## 使用技術

- Google Apps Script (V8)
- BigQuery API v2
- Vanilla JavaScript / HTML / CSS
- Font Awesome 6.5

## BigQuery データソース

| 項目 | 値 |
|---|---|
| プロジェクト ID | `itdc-wdr` |
| データセット | `daily_reports` |
| テーブル | `reports` |

## デプロイ手順

1. [Google Apps Script](https://script.google.com) で新規プロジェクトを作成
2. 本リポジトリの各ファイルをコピー（または clasp で push）
3. BigQuery API を有効化（高度なサービス）
4. 「デプロイ」→「新しいデプロイ」→ 種類: **ウェブアプリ**
5. 実行ユーザー: **自分**、アクセス: **自分のみ** に設定してデプロイ
